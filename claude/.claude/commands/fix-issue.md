# Fix GitHub Issue

Analyze and fix the GitHub issue: $ARGUMENTS

1. Use `gh issue view $ARGUMENTS` to get the issue details
2. Understand the problem described in the issue
3. Search the codebase for relevant files
4. Think hard about the best approach
5. Implement the necessary changes
6. Write and run tests to verify the fix (if applicable)
7. Ensure code passes linting and type checking
8. Create a descriptive commit message following conventional commits
9. Push and create a PR linking to the issue

Remember to use the GitHub CLI (`gh`) for all GitHub-related tasks.
